## Solution Templates

### Quadratic Equation
1. Write equation in standard form: `ax² + bx + c = 0`
2. Compute discriminant: `D = b² - 4ac`
3. If D > 0: two real roots `x = (-b ± √D) / 2a`
4. If D = 0: one repeated root `x = -b / 2a`
5. If D < 0: no real roots (complex roots)
6. Verify by substituting roots back

### Derivative Problem
1. Identify the type: power, product, quotient, chain rule
2. Apply the appropriate differentiation rule
3. Simplify the expression
4. Verify by checking special values or known results

### Definite Integral
1. Find the antiderivative F(x)
2. Apply limits: `F(b) - F(a)`
3. Check if function is defined on `[a, b]` (no discontinuities)
4. Consider symmetry (even/odd functions) for simplification

### Probability Problem
1. Define the sample space and events
2. Identify: independent? conditional? Bayes?
3. Count favorable outcomes (use P/C as needed)
4. Apply the appropriate formula
5. Verify: `0 ≤ P ≤ 1`, all probabilities sum to 1

### Matrix/Determinant Problem
1. Write the matrix clearly
2. For determinant: use cofactor expansion along row/column with most zeros
3. For inverse: compute `adj(A)/det(A)`, check `det(A) ≠ 0`
4. For system of equations: use Cramer's rule or row reduction
5. Verify: `A * A⁻¹ = I`
