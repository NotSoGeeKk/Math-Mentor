## Algebra Identities

### Standard Identities
- `(a + b)^2 = a^2 + 2ab + b^2`
- `(a - b)^2 = a^2 - 2ab + b^2`
- `(a + b)(a - b) = a^2 - b^2`
- `(a + b + c)^2 = a^2 + b^2 + c^2 + 2ab + 2bc + 2ca`

### Quadratic Formula
For `ax^2 + bx + c = 0`:
`x = (-b +- sqrt(b^2 - 4ac)) / (2a)`

### Binomial Theorem
`(x + y)^n = sum_{k=0 to n} [C(n,k) * x^(n-k) * y^k]`
