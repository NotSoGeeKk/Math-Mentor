## Calculus Applications

### Finding Maxima and Minima
1. Find `f'(x) = 0` → critical points
2. Second derivative test:
   - `f''(c) > 0` → local minimum at c
   - `f''(c) < 0` → local maximum at c
   - `f''(c) = 0` → test is inconclusive, use first derivative test

### First Derivative Test
At critical point c:
- `f'` changes from + to − → local maximum
- `f'` changes from − to + → local minimum

### Increasing / Decreasing Functions
- `f'(x) > 0` on interval → f is increasing
- `f'(x) < 0` on interval → f is decreasing

### Rate of Change
`dy/dx` at `x = a` gives the instantaneous rate of change of y w.r.t. x at that point.

### Tangent and Normal Lines
At point `(x₀, y₀)` on curve `y = f(x)`:
- Slope of tangent: `m = f'(x₀)`
- Tangent line: `y - y₀ = m(x - x₀)`
- Normal line (perpendicular): `y - y₀ = (-1/m)(x - x₀)`

### Optimization Strategy
1. Define variables and constraint equations
2. Express quantity to optimize as function of one variable
3. Find critical points using derivative = 0
4. Verify it's a max or min using second derivative test
5. Check boundary values if domain is closed
