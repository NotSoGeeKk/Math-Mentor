## Common JEE Math Mistakes

### Algebra
- Forgetting that `√(x²) = |x|`, not just `x`
- Cancelling terms without checking if denominator could be zero
- Misapplying the quadratic formula: forgetting the `±` sign
- Using `(a+b)² = a² + b²` instead of `a² + 2ab + b²`

### Calculus
- Confusing `d/dx [f(x)·g(x)]` with `f'(x)·g'(x)` — must use product rule
- Forgetting the chain rule when differentiating composite functions
- Ignoring `+ C` in indefinite integrals
- Applying L'Hôpital's rule when limit is not indeterminate

### Probability
- Adding probabilities of non-mutually-exclusive events without subtracting intersection
- Confusing `P(A|B)` with `P(B|A)`
- Using permutations when order doesn't matter (should use combinations)

### Linear Algebra
- Assuming `AB = BA` (matrix multiplication is not commutative)
- Forgetting that `det(AB) = det(A) * det(B)`, not `det(A+B)`
- Applying Cramer's rule when determinant is zero

### General
- Not checking domain constraints (log of negative, sqrt of negative, division by zero)
- Dropping absolute value signs in logarithmic results
- Skipping verification of the answer by substitution
