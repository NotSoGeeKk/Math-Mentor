## Logarithms

### Definition
`log_b(x) = y` means `b^y = x`, where `b > 0, b ≠ 1, x > 0`.

### Fundamental Rules
- `log_b(xy) = log_b(x) + log_b(y)`
- `log_b(x/y) = log_b(x) - log_b(y)`
- `log_b(x^n) = n * log_b(x)`
- `log_b(1) = 0`
- `log_b(b) = 1`

### Change of Base Formula
`log_b(x) = log_a(x) / log_a(b)`
Special case: `log_b(x) = ln(x) / ln(b)`

### Common Identities
- `b^(log_b(x)) = x`
- `log_b(x) = 1 / log_x(b)`
- `log_a(b) * log_b(c) = log_a(c)` (chain rule for logs)

### Natural Logarithm
- `ln(e) = 1`, `ln(1) = 0`
- `d/dx ln(x) = 1/x`
- `∫ 1/x dx = ln|x| + C`

### Domain Constraints
`log_b(x)` is defined only for `x > 0`. Always check domain in JEE problems.
